# 🎌 Anime_Searcher

¡Bienvenido a **Anime_Searcher**! Un buscador minimalista, moderno y veloz de personajes de anime. Este proyecto nació como una herramienta para fans que buscan información técnica de sus personajes favoritos de cualquier serie, desde Bleach hasta los estrenos más recientes.

---

## 🚀 Vista Previa
> **¡Pruébalo en vivo aquí!** -> [[LINK_DE_VERCEL_AQUÍ_MAÑANA](https://anime-searcher-five.vercel.app)]

## ✨ Características Principales
- 🔍 **Búsqueda Avanzada:** Encuentra personajes de cualquier anime usando la API de Jikan (MyAnimeList).
- 💎 **Diseño Minimalista:** Interfaz tipo "Glassmorphism" con animaciones suaves y centradas.
- 📱 **100% Responsivo:** Optimizado para PC, Tablets y Móviles.
- ⚡ **Manual Search:** Solo realiza peticiones cuando tú lo decides (Enter o Click), optimizando el consumo de datos.

## 🛠️ Stack Tecnológico
Este proyecto fue construido utilizando las mejores prácticas de desarrollo moderno:
- **React 18** (Vite)
- **TypeScript** (Tipado estricto para evitar errores)
- **CSS3 Moderno** (Custom Properties, Flexbox y Grid)
- **Axios** (Peticiones HTTP eficientes)
- **Clean Architecture** (Separación de lógica en Hooks y Actions)
